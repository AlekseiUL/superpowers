## Overview
Building high-quality iOS and Android apps using React Native or Flutter.

## When to Use
- Mobile app development
- Store submission
- Native modules

## Core Principles
- **Offline-First**: Apps must work without internet.
- **Performance**: 60fps is non-negotiable.
- **Platform Guidelines**: Follow Human Interface Guidelines (iOS) and Material Design (Android).

## Workflow
1. **Setup** (Expo/Flutter CLI).
2. **Development** (Simulators).
3. **Build** (EAS/Fastlane).
4. **Publish** (App Store Connect/Google Play Console).

## Checklist
- [ ] Does it handle offline state?
- [ ] Are touch targets large enough (44px+)?
- [ ] Is dark mode supported?
